# Sanity Studio テストデータ入力手順書

このドキュメントは、Sanity Studio に手動でテストデータを入力する手順をまとめたものです。

## 前提条件

- 開発サーバーが起動していること (`npm run dev`)
- http://localhost:3000/studio にアクセス可能であること
- Sanity にログイン済みであること

---

## 1. サイト設定（siteSettings）

### 手順
1. サイドバーから「サイト設定」または「Site Settings」を選択
2. 以下の値を入力：

| フィールド | 値 |
|-----------|-----|
| キャッチコピー | AIで、映像の常識を変える |
| サブキャッチコピー | AI動画・AI漫画・デザインで、あなたのビジネスを加速 |
| 代表動画URL | https://www.youtube.com/watch?v=dQw4w9WgXcQ |
| サイトタイトル | KENJIROU Portfolio |
| サイト説明文 | AI動画制作・AI漫画・デザインサービスのポートフォリオサイト |

3. 「Publish」ボタンをクリック

---

## 2. プロフィール（profile）

### 手順
1. サイドバーから「プロフィール」または「Profile」を選択
2. 基本情報を入力：

| フィールド | 値 |
|-----------|-----|
| 名前 | 林 憲二郎 |
| 英語名 | Kenjirou Hayashi |
| 自己紹介文 | デザイン歴12年。AIを活用した動画・漫画制作を行っています。キャラクターの一貫性を保ったAI動画制作が強みです。 |

3. 「強み」セクションで「Add item」を3回クリックし、以下を入力：

**強み1:**
- タイトル: `キャラクター一貫性`
- 説明: `AIでもキャラクターの一貫性を保った動画制作が可能`

**強み2:**
- タイトル: `デザイン経験`
- 説明: `12年のデザイン経験を活かした高品質な映像制作`

**強み3:**
- タイトル: `スピード納品`
- 説明: `AIの活用により、短納期での制作に対応`

4. 「Publish」ボタンをクリック

## 完了確認

すべてのデータ入力が完了したら、以下を確認してください：

### 各ドキュメントの保存件数
- [ ] サイト設定: 1件（Published）
- [ ] プロフィール: 1件（強み3件含む、Published）
- [ ] サービス: 3件（すべてPublished）
- [ ] 制作実績: 2件（すべてPublished）

### 確認方法
1. Sanity Studio のサイドバーで各ドキュメントタイプをクリック
2. 一覧に正しい件数のドキュメントが表示されることを確認
3. 各ドキュメントを開いて、ステータスが「Published」になっていることを確認

---

## トラブルシューティング

### フィールドが見つからない場合
- スキーマ定義を確認: `sanity/schemas/` ディレクトリ内のファイルを参照
- フィールド名が異なる可能性があります（英語名 vs 日本語名）

### Publish ボタンが押せない場合
- 必須フィールドがすべて入力されているか確認
- エラーメッセージが表示されていないか確認

### データが保存されない場合
- ブラウザのコンソールにエラーが出ていないか確認
- 開発サーバーが正常に動作しているか確認
- Sanity Studio の接続状態を確認

---

## 次のステップ

テストデータの入力が完了したら、Phase 3（フロントエンド実装）に進むことができます。

入力したデータは、以下のページで表示されるようになります：
- トップページ: サイト設定、プロフィール、トップ表示の制作実績
- サービス一覧・詳細ページ
- 制作実績一覧・詳細ページ
